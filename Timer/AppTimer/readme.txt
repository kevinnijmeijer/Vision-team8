Je kan simpel een applicatie timen met deze applicatie. Sleep een applicatie op de AppTimer.exe of run een command in de command line. Bijvoorbeeld om te timen hoe lang je de rekenmachine gebruikt:

  AppTimer calc.exe
  
Of als je zelf een programma hebt gemaakt die 'my.exe' heet die enkele argumenten mee krijgt:
  
  AppTimer my.exe -t 100 -v 102
